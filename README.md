# Memory Game

A React-based memory card matching game with customizable grid size and move limits.

## Features

- Adjustable grid size (up to 10x10)
- Optional move limit setting
- Score tracking
- Responsive design
- Game settings panel
- Win/lose conditions

## Technologies Used

- React
- Tailwind CSS
- Vite
- ESLint
